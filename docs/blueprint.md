# **App Name**: RoomBase

## Core Features:

- User Authentication: Secure signup and login for Room Finders and Room Owners using Supabase Auth.
- Room Search: Enable Room Finders to search rooms by location with filtering by price range, property type, and tenant preference.
- Room Details View: Display detailed information for each room, including images, title, location, rent price, property type, tenant preference, and owner contact number.
- Room Listing Management: Allow Room Owners to add new room listings with multiple images, and to edit or delete their listings.
- Image Upload: Allow Room Owners to upload multiple images for their listings.

## Style Guidelines:

- Primary color: Warm neutral taupe (#D9CDBA) for a sense of comfort and home.
- Background color: Very light, desaturated beige (#F5F3F0).
- Accent color: Soft gold (#E6C571) to highlight important elements and CTAs.
- Headline font: 'Playfair', a modern serif with an elegant, fashionable feel; for longer text, use 'PT Sans' for body
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, outlined icons related to property types and amenities.
- Implement a responsive grid layout for room listings that adapts to different screen sizes.
